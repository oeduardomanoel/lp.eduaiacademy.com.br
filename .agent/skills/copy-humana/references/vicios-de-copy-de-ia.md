# Vicios de Copy de IA

Lista negativa para evitar texto com cara de maquina.

## Sinais classicos de copy artificial

1. Abertura generica, sem contexto real do publico.
2. Promessa inflada sem mecanismo ou prova.
3. Excesso de adjetivos, pouco conteudo concreto.
4. Frases cliche repetidas em varios paragrafos.
5. Estrutura engessada que ignora objecoes reais.
6. CTA vazio, sem motivo claro para agir agora.

## Frases cliche para reprovar e reescrever

1. No mundo atual.
2. Em um mundo onde.
3. Descubra o poder de.
4. Solucao revolucionaria.
5. Transforme sua vida em poucos passos.
6. Nao apenas X, mas tambem Y, sem especificidade.
7. Segredo que ninguem te contou.

## Negativos operacionais

Reprovar a copy se houver qualquer item abaixo:

1. Frase bonita sem dado, prova ou exemplo.
2. Beneficio sem mecanismo.
3. Dor sem linguagem do cliente.
4. Texto que poderia vender qualquer coisa para qualquer pessoa.
5. Paragrafo que nao move a decisao para o proximo passo.

## Substituicoes para soar humano

1. Em vez de falar de forma vaga, use situacao observavel.
2. Em vez de superlativo, use numero, prazo ou comparacao concreta.
3. Em vez de frase de efeito, use objecao real e resposta objetiva.
4. Em vez de tom professoral, use tom de conversa util.
5. Em vez de bloco longo monotono, varie ritmo e intencao.

## Detector rapido de qualidade

Se o texto nao responder com clareza a estas perguntas, reescreva:

1. Para quem isso foi escrito?
2. Qual problema especifico isso resolve?
3. Qual mecanismo torna a promessa plausivel?
4. Qual prova sustenta a afirmacao?
5. Qual acao unica o leitor deve tomar agora?
