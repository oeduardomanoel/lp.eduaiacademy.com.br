# Autores e Principios

Este arquivo consolida principios praticos dos autores solicitados para orientar a escrita de copy.

Use como biblioteca de estrategia, nao como imitacao literal de estilo pessoal.

## Russell Brunson

Base principal: arquitetura de funil e estrutura de oferta.

Principios para aplicar:

1. Hook Story Offer: gancho, historia, oferta.
2. Value Ladder: oferta de entrada, ascensao de ticket, continuidade.
3. Oferta com pilha de valor clara, para reduzir friccao de compra.
4. Mensagem orientada a sonho do cliente, com caminho simples para o proximo passo.

Quando usar:

1. Pagina de vendas.
2. Sequencia de funil.
3. Oferta de entrada e upsell.

## Todd Brown

Base principal: mecanismo unico e campanha de aquisicao previsivel.

Principios para aplicar:

1. Vender sem parecer que esta vendendo.
2. Liderar com problema e mecanismo, nao com hype de produto.
3. Educacao estrategica antes do pitch.
4. Encadear argumentos para reduzir resistencia e elevar desejo.

Quando usar:

1. VSL.
2. Webinar.
3. Campanha evergreen.

## Ben Settle

Base principal: email com voz forte e venda recorrente.

Principios para aplicar:

1. Frequencia consistente de contato.
2. Persona editorial clara, com opiniao e posicionamento.
3. Historia curta que termina em oferta ou acao.
4. CTA simples e direto, sem excesso de opcoes.

Quando usar:

1. Newsletter de vendas.
2. Sequencia de relacionamento.
3. Reengajamento de lista fria.

## Paulo Maccedo

Base principal: storyselling e resposta direta com linguagem brasileira.

Principios para aplicar:

1. Narrativa emocional com utilidade comercial.
2. Problema concreto, conflito, virada, solucao aplicavel.
3. Estrutura orientada a venda, sem perder humanidade.
4. Clareza de mensagem para publico amplo no Brasil.

Quando usar:

1. Carta de vendas.
2. Video de venda.
3. Conteudo de aquecimento para oferta.

## Bettina Rudolph

Base principal: transformacao com liberdade de tempo e escolha.

Principios para aplicar:

1. Contraste entre rotina de sobrecarga e vida com autonomia.
2. Promessa ancorada em ganho de tempo, energia e direcao.
3. Tom de mentoria proxima, com acolhimento e decisao.
4. Comunidade e pertencimento como reforco de seguranca para compra.

Quando usar:

1. Ofertas para publico feminino.
2. Programas de educacao e transformacao de rotina.
3. Narrativas de libertacao do caos operacional.

## Combinacoes praticas

1. Captacao fria: Brunson mais Brown.
2. Email diario: Settle mais Maccedo.
3. Oferta de liberdade: Bettina mais Brown.
4. Pagina longa de conversao: Brunson mais Maccedo.
